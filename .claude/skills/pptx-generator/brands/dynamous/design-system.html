Glass Cards

The signature UI pattern for Dynamous. Frosted glass effect over dark backgrounds.

Default Glass Card

Background: rgba(0, 0, 0, 0.4)
Backdrop blur: 16px
Border: rgba(255, 255, 255, 0.1)
Border radius: 12px

Highlighted Card

Same base with blue border and subtle glow for emphasis.

Announcement Card

Green border variant for announcements and success states.

CSS Implementation

.glass-card { background: rgba(0, 0, 0, 0.4); backdrop-filter: blur(16px); -webkit-backdrop-filter: blur(16px); border: 1px solid rgba(255, 255, 255, 0.1); border-radius: 12px; }

Glow Effects

Signature emphasis effects. Use sparingly — if everything glows, nothing does.

Box Glow
box-shadow: 0 0 15px rgba(59, 130, 246, 0.5)
Text Glow
text-shadow: 0 0 10px rgba(59, 130, 246, 0.5)
Combined
Text + Box

CSS Implementation

.blue-glow { box-shadow: 0 0 15px rgba(59, 130, 246, 0.5), 0 0 30px rgba(59, 130, 246, 0.3); } .text-glow { text-shadow: 0 0 10px rgba(59, 130, 246, 0.5), 0 0 20px rgba(59, 130, 246, 0.3); }
